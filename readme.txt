WP 2 Step Facebook Authentication provides an extra layer of facebook authentication to Wordpress account.

== Description ==
WP 2 Step Facebook Authentication is a lightweight stable plugin that provides an extra security layer of facebook authentication to Wordpress account.
It comes with minimal configuration and works with the latest version of WordPress and Facebook SDK.

**WP 2 Step Facebook Authentication provides the following features**

- Add extra authentication layer of Facebook to Wordpress account.
- Stop spam users and subscribers from login.
- Role based authentication.
- Very lightweight. 
- User friendly.  
- Easy to configure.
- Best support.
- Compatible with latest version of Wordpress.
- Free updates for lifetime.

== Installation ==
1. Upload "wp2step-fb-authentication" folder to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Go to "2 Step FB Setting" at left side menu bar.
1. Provide Facebook AppID and App secret key at the plugin's backend section.
1. Save Changes, that's it and Done.

3. On clicking Facebook login button, it shows the login pop from Facebook.
4. On providing valid Facebook credentials, it pull basic informantion like profile image, name and email id.
   On clicking 'Yes this is me', it compares the Wordpress user email id with Facebook email id. If it is matched successfully, it redirects to admin. Else it gives same error message.
